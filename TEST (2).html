Basic Definitions

Variable: Used to store information (like numbers or text).

Data Type: Tells what kind of data a variable stores (like int, char).

Loop: Repeats a set of instructions (like for loop or while loop).

Example Programs

Hello World

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

This program prints "Hello, World!" on the screen.

Add Two Numbers

#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}

This program adds 10 and 20, and shows the result.
